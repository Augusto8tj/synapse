# Synapse: Mentes Criativas

## Visão Geral

Synapse é uma plataforma educacional gamificada projetada para inspirar e capacitar crianças no aprendizado de programação, lógica, criatividade e colaboração. Utilizando inteligência artificial de ponta, o Synapse oferece experiências de aprendizado personalizadas, adaptando-se às necessidades e interesses individuais de cada criança.

## Objetivos Principais

*   **Engajamento Ativo:** Proporcionar um ambiente de aprendizado lúdico e interativo através de jogos e atividades.
*   **Desenvolvimento de Habilidades:** Ensinar conceitos fundamentais de programação, lógica, física, engenharia e artes.
*   **Estímulo à Criatividade:** Oferecer ferramentas robustas como o Laboratório de Invenções e o Estúdio do Contador de Histórias.
*   **Fomento à Colaboração:** Incentivar o trabalho em equipe através de projetos conjuntos.
*   **Aprendizado Personalizado:** Utilizar IA para adaptar o conteúdo, o ritmo e a tutoria às necessidades de cada usuário.

## Recursos Principais

O Synapse oferece uma variedade de módulos interativos:

*   **Onboarding Dinâmico com IA:** Um processo inicial adaptativo que utiliza minijogos para mapear as aptidões e interesses de cada criança, personalizando sua jornada de aprendizado desde o primeiro momento.
*   **Mentor de IA Personalizado:** Cada criança interage com um avatar de IA que funciona como um companheiro de aprendizado, oferecendo questionamentos socráticos, sugestões de projetos desafiadores e reconhecimento individual do progresso.
*   **Laboratório de Invenções (Perfil 1):** Um sandbox 3D onde as crianças podem projetar, construir e simular máquinas, explorando princípios de física e lógica de forma prática.
*   **Estúdio de Programação (Perfil 1):** Uma plataforma que evolui da programação visual em blocos para linguagens como Python e JavaScript, permitindo a criação de jogos, automações e arte generativa.
*   **Estúdio do Contador de Histórias (Perfil 2):** Um espaço criativo para desenvolver personagens e narrativas, com o auxílio da IA para sugestões, geração de imagens a partir de descrições (texto-para-imagem) e criação de trilhas sonoras adaptativas.
*   **Playground de Lógica (Perfil 2):** Um jogo de quebra-cabeça baseado em física onde as crianças constroem estruturas para resolver desafios, aprendendo sobre causa e efeito, gravidade e engenharia básica.
*   **Projetos Colaborativos:** A IA pode gerar desafios que incentivam a colaboração entre as crianças, como a criação de um jogo de aventura interativo que integre recursos de imagem, som, vídeo e atividades de colorir.

## Configuração e Execução

Este projeto é uma aplicação Next.js. Para configurá-lo e executá-lo localmente, siga estes passos:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/Augusto8tj/synapse.git
    cd synapse
    ```
2.  **Instale as dependências:**
    ```bash
    npm install
    ```
3.  **Configure as variáveis de ambiente:**
    Crie um arquivo `.env.local` na raiz do projeto e adicione suas credenciais do Firebase. Consulte a documentação do Firebase para obter detalhes sobre como configurar seu projeto.
4.  **Execute o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
5.  Acesse a aplicação em seu navegador no endereço `http://localhost:3000`.
